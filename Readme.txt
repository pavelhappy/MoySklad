Тестовое задание: 

Требования: ​
1. Задание должно быть выполнено на языке Java. 
2. Для сборки использовать любую из распространенных систем сборки (maven, ant, 
gradle и т.д.). 
3. Исходный код должен быть выложен на github.com или bitbucket.com и снабжён 
инструкциями по сборке и запуску. 
 
Задание: ​
Разработать программу на языке Java для обработки файла, в котором задаются 
зависимости сущностей друг от друга. Каждая сущность представлена своим 
уникальным идентификатором в виде целого числа. Зависимости задаются 
строкой в текстовом файле, содержащем произвольное количество строк формата: 
<id сущности> <id сущности от которой она зависит>. Файл читать из STDIN. Для 
прочитанных сведенй о связях между сущностями необходимо найти циклические 
зависимости и вывести их в STDOUT в виде id сущностей через пробел. 
 
Пример файла: ​
1 2 
2 1 
3 4 
5 6 
6 5 
 
Ожидаемый вывод в STDOUT: ​
1 2 1 
5 6 5 

######################################################

Инструкция по сборке и запуску:
1) Для сборки используется Maven. Установить можно с https://maven.apache.org/
2) Перейдите в директорию с файлом pom.xml  и выполните команду в консоли: mvn package
3) После окончания сборки в папке target/ появится jar-файл.
4) Запустите jar-файл из консоли командой: java -jar НазваниеФайла.jar
